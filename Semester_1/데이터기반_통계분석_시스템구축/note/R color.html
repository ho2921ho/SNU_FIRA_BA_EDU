R colors

  • 16진법 표기를 사용한다. (0 ~ 9, A ~ F)
  • 기본적으로 빨간색, 초록색, 파란색을 지정하며, 16진법으로 2개씩 표현한다. 즉, 기본 색상 별로 각각 256개로 표현 할 수 있다. (#rrggbb)
  • 그러므로 256×256×256=224256×256×256=224개의 색상이 표현된다. : 24 bit colors.
  • 또한, 24 bit colors에 투명도(256 level)를 추가하면 32 bit color system이 된다.(#rrggbbaa)

rgb function

  • rgb(red, green, blue, maxColorValue, alpha)
  • red, green, blue : 기본으로 0 ~ 1 사이의 값을 사용한다.
  • maxColorValue = 255 으로 세팅하게 되면 0 ~255 사이의 값으로 사용할 수 있다.
  • alpha : opacity(투명도)
rgb(10, 4, 23, maxColorValue = 255, alpha = 10)
## [1] "#0A04170A"
col2rgb('lightblue')
##       [,1]
## red    173
## green  216
## blue   230

hue color system

  • 색상들은 콘 모양으로 볼수 있다.
  • 바깥쪽으로 갈수록 채도가 높은 색이다.
  • 위쪽으로 갈수록 밝기는 증가하게 된다.

hcl function

  • hcl(h, c, l, alpha)
  • h: colors(색상), c: saturation(채도), l: brightness(밝기), alpha: opacity(투명도)
  • h∈[0,360]∈[0,360]: 0~120(red), 120~240(green), 240~360(blue)
  • l ∈[0,100]∈[0,100], alpha ∈[0,1]∈[0,1]
  • c의 범위는 h와 l의 값에 따라 변한다. (즉, 색상은 채도, 밝기에 의존한다.)
hcl(h = 0, c = 35, l = 85, alpha = 0.1)
## [1] "#FFC5D01A"

hsv function

  • hsv(h, s, v, alpha)
  • h: color(색상), s: satulation(채도), v: brightness(밝기), alpha: opacity(투명도)
  • h, s, v, alpha ∈[0,1]∈[0,1]
hsv(0.3, 0.5, 0.1, alpha = 0.4)
## [1] "#0F1A0D66"

grDevices, colorRamps Packages

  • rainbow colors는 hsv color system을 기반으로 한다.
  • n (색상의 수), s (채도), v (명도), start (h의 시작값), end (h의 끝값), alpha (투명도)
  • 시작값과 끝값의 범위: 0 ~ 1
rainbow(5, s = 0.4, v = 0.3, start = 0, end = 0.05, alpha = 1)
## [1] "#4D2E2EFF" "#4D302EFF" "#4D322EFF" "#4D352EFF" "#4D372EFF"

RColorBrewer package

  • 이 패키지를 사용하면 R에서 제공되는 palette들을 사용할 수 있다.
  • sequential palettes : Blues, BuGn, BuPu, …, YlOrBr, YlOrRd
if(!require(RColorBrewer)){install.packages("RColorBrewer"); library(RColorBrewer)}
## Loading required package: RColorBrewer
brewer.pal(4, 'Blues')
## [1] "#EFF3FF" "#BDD7E7" "#6BAED6" "#2171B5"
